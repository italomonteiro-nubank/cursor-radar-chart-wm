## Radar Chart (Chart.js) — Figma `1324:17002`

- **Demo**: abra `[snippet.html](snippet.html)`.
- **Scores**: cada dimensão aceita **1–100** (onde **1 = centro** e **100 = vértice**).
- **Dimensões** (ordem do design): Diversificação, Liquidez, Retorno, Eficiência, Proteção.

### Rodando localmente
Como o `snippet.html` usa `type="module"`, sirva a pasta via HTTP:

```bash
cd radar
python3 -m http.server 8000
```

Depois acesse `http://localhost:8000/snippet.html`.
